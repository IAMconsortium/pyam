.. _install:

Install
*******

Via Conda
~~~~~~~~~

.. code-block:: bash

    conda install -c conda-forge pyam

Via Pip
~~~~~~~

.. code-block:: bash

    pip install pyam-iamc

From Source
~~~~~~~~~~~

.. code-block:: bash

    pip install -e git+https://github.com/IAMconsortium/pyam.git#egg=pyam
