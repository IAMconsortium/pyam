.. _install:

Install
*******

Via Conda
~~~~~~~~~

.. code-block:: bash

    conda install -c conda-forge pyam

Via Pip
~~~~~~~

.. code-block:: bash

    pip install pyam-iamc

From Source
~~~~~~~~~~~

.. code-block:: bash

    git clone https://github.com/IAMconsortium/pyam && cd pyam
    pip install -r requirements.txt && python setup.py install
