.. _install:

Install
*******

Via Conda
~~~~~~~~~

.. code-block:: bash

    conda install -c pyam

Via Pip
~~~~~~~

.. code-block:: bash

    pip install pyam-iamc

From Source
~~~~~~~~~~~

.. code-block:: bash

    pip install -r requirements.txt && python setup.py install
